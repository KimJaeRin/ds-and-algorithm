## 6일차
- 파이썬 자료구조/알고리즘
    - 재귀호출
    - 정렬
        - 선택 정렬 Selection Sort - 최소값을 찾아서 맨 앞으로 O(n^2)
        - 삽입 정렬 Insertion Sort - 기준값 기준 앞뒤로 보내는 정렬 O(n^2)
        - 버블 정렬 Bubble Sort - 기준값 기준으로 제일 긑 값을 뒤로 보내는 정렬 O(n^2)
        - 퀵 정렬   Quick Sort - 기준값 작은값 그룹/큰값그룹을 분리한 뒤 다시 정렬 재귀호출 O(nlogn)
